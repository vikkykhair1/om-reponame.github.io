  1. Player who sets a challenge will share a room id/room code with his/her opponent.
  2. Player can choose no of coin he wish to play with for match with opponent and then start game accordingly

    • if opponent is not available for same amount or type match then game will be cancelled
    • if match is not started and both parties doesn't want to play, select ‘Cancel’ option.
